Профессия Веб-разработчик

Вы с нуля научитесь создавать полноценные сайты и веб-приложения. Освоите HTML, CSS, JavaScript, ReactJS и Git, познакомитесь с фреймворками и станете востребованным разработчиком. Даже если до этого были не знакомы с программированием.

Веб-разработчик создаёт сайты, сервисы и приложения, которыми мы ежедневно пользуемся. Он разрабатывает интернет-магазины, онлайн-банки поисковики, карты и почтовые клиенты. Веб-разработчик проектирует внешний вид сайта — фронтенд и программирует сервисную часть — бэкенд.


Спрос на веб-разработчиков остаётся высоким — прибыль бизнеса напрямую зависит от того, насколько удобно клиентам пользоваться сайтом компании.

Кому подойдёт этот курс

Новичкам в веб-разработке

С нуля изучите основы программирования на JavaScript и PHP. Научитесь верстать сайты на HTML и CSS. Создадите первые проекты для портфолио.

Начинающим разработчикам

Прокачаетесь в работе с самыми популярными языками программирования Изучите фреймворки Vue, React или Angular и расширите круг своих компетенций в кодинге.

IT-специалистам

Научитесь программировать на JavaScript и PHP, будете создавать проекты на фреймворке Laravel. Сможете поменять направление и стать востребованным веб-разработчиком.

Чему вы научитесь

Верстать сайты на HTML и CSS

Вы научитесь создавать красивые сайты, которые будут одинаково хорошо выглядеть на любых устройствах - начиная экранами смартфонов и заканчивая широкими экранами телевизоров

Программировать на JavaScript и React

Освоите один из самых востребованных языков — JavaScript, с помощью которого можно программировать всё что угодно: от простеньких алгоритмов до 3d моделирования и игр

Работать с JavaScript-фреймворками

С помощью фреймворков будете легко собирать веб-приложения из отдельных компонентов, как из деталей конструктора. Вы будете изучать самые популярные из них - React и JQuery.

Реальному процессу разработки

На протяжении всего обучения вы будете выполнять практические задания, а свой прогрессом сохранять при помощи системы контроля версий Git - наиважнейшего инструмента любого разработчика

Вводные уроки

Знакомство с веб-разработкой

На вводных уроках вы получити базовые знания о программировании и веб-разработке, научитесь использовать среду разработки и использовать системы управлениями версий.

Теория

Получаете знания

Курс состоит из тематических лекций, включающих в себя как письменные материалы, так и интеравтивные медиа. Все лекции создавались с учетом международных практик и советов по изучению программирования.

Практика

Выполняете задания

Мы уверены, что только через практику возможно отрабатывать навыки. Поэтому после теории вас ждёт практическая работа над проектами. Все задачи приближены к реальным - их можно с гордостью разместить в портфолио.

Обратная связь

Работаете с куратором

Проверкой заданий занимаются кураторы. Они помогут с трудными заданиями, направят в правильную сторону и подскажут, как можно улучшить вашу работу.
